Документи HTML

Всі HTML - документи повинні починатися з оголошення типу документа: !DOCTYPE html.

Сам HTML-документ починається html і закінчується на /html.

Видима частина HTML -документа знаходиться між body і /body.

HTML не чутливий до регістру

Стандарт HTML не вимагає використання тегів у нижньому регістрі,
але W3C рекомендує малі літери в HTML та вимагають малі регістри для більш жорстких типів документів, таких як XHTML.

ми завжди використовуємо імена тегів у нижньому регістрі.

Щоб отримати повний список усіх доступних HTML -тегів, відвідайте наш довідник HTML -тегів .

Заголовки HTML

Заголовки HTML визначаються за допомогою тегів h1 до h6.

h1визначає найважливіший заголовок.

h6визначає найменш важливий заголовок.
